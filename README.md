# Praneeth Narayanan - Portfolio Website

A modern, responsive portfolio website showcasing my work as an AI Engineer and Full Stack Developer.

## 🚀 Features

- Dark minimalist theme with neon accents
- Terminal-style hero section with typing animation
- Animated skill bars and project cards
- Glassmorphism design elements
- Smooth scroll animations
- Fully responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Inter, JetBrains Mono)

## 📂 Structure

```
├── index.html    # Main HTML file
├── styles.css    # All styling
├── script.js     # Animations and interactions
└── README.md     # This file
```
